# Ottawa Housing Suitability Assessment

## Project Overview:
This project identifies suitalbe urban housing areas in Ottawa using spatial analysis techniques.

## Study Area:
Ottawa, Ontario, Canada

## Data Sources:
The spatial analysis used three publicly available datasets obtained from the city of Ottawa open data portal: 
1. Ottawa Secondary Schools:
Point locations representing secondary schools within the City of Ottawa. This dataset was used to identify residential areas with convenient access to educational facilities. 
2. O-Train Stations:
Point locations of O-Train stations within Ottawa's light rail transit network. This dataset was used to evaluate accessibility to public transportation.
3. Public Libraries:
Point locations of Ottawa Public Library branches. Libraries were included as an indicator of access to community services and public resources.

All datasets were downloaded from the city of Ottawa Open Data Portal and processed in ArcGIS Pro for spatial analysis.

## Methodology:
Buffer analysis and overlay analysis were used to identify areas with access to key urban services.

## Results
Areas near schools, transit, and libraries were identified as the most suitalbe locations for housing development.
