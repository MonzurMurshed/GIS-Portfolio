# Data Sources

This project uses publicly available datasets.

## Raw Data (`raw/`)

The original datasets used in this project:

- NOAA/NSIDC Sea Ice Concentration Climate Data Record (CDR) Version 6
- National Geospatial-Intelligence Agency (NGA) Arctic Sea Routes
- Natural Earth Admin 0 Countries

## Processed Data (`processed/`)

Processed outputs created in ArcGIS Pro, including corridor extraction and sea ice concentration change rasters.

## Notes

Raw datasets are not included in this repository to reduce repository size. They can be downloaded from the original sources listed below.
